# Fast Food Marketing Campaign A/B Testing Analysis

This project evaluates **three marketing promotions** from a fast-food chain to identify which strategy generates the highest sales uplift.

## Objective
Compare the effectiveness of three promotion types and recommend the best-performing strategy for 2025.

## Key Findings
- **Promotion 1 and Promotion 3** showed the highest sales uplift.
- Statistical tests confirmed that these differences are **significant**.
- Promotion 2 performed worse and is not recommended.

## Conclusion
**Implement Promotion 1 or Promotion 3** as the main strategy for the upcoming year.

## Future Considerations
- Test over longer periods for more stable results.
- Segment results by customer demographics.
- Evaluate profitability, not just sales volume.
- Consider external influences like seasonality and competition.

## Tools Used
Python, Pandas, Seaborn, Matplotlib, SciPy
